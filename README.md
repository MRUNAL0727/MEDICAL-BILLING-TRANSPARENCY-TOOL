# MEDICAL-BILLING-TRANSPARENCY-TOOL

## 📌 Project Overview
The **Medical Billing Transparency Tool** is a solution designed to promote transparency in hospital billing across India. This project analyzes variations in healthcare costs across hospitals, cities and medical procedures, enabling stakeholders to better understand cost distributions, detect anomalies and evaluate billing fairness using statistical and machine learning techniques. By combining exploratory data analysis, hypothesis testing, predictive modeling and interactive visualization, the tool transforms raw billing data into meaningful financial insights.

## 🎯 Project Goal
To enhance transparency in healthcare pricing by:
1. Identifying cost disparities across hospitals and cities
2. Analyzing billing patterns and financial trends
3. Applying statistical validation techniques
4. Building a predictive model to estimate final hospital bills
5. Visualizing insights through an interactive dashboard

## 🔍 Key Features
📊 Interactive Power BI Dashboard for dynamic data exploration
📁 Custom simulated dataset of 120 hospital billing records
🧪 Statistical hypothesis testing (t-test, ANOVA, correlation analysis)
📈 Linear Regression model to predict final hospital bill
🔍 Exploratory Data Analysis (EDA) including summary statistics, city-wise analysis, correlation matrices and cost trend visualization

## 🧾 Dataset Description
The dataset contains structured hospital billing information with the following fields:
1. Hospital Name
2. City
3. State
4. Procedure Cost
5. Room Charge
6. Lab Fee
7. Insurance Discount
8. Final Bill
